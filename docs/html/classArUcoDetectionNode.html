Detailed Description

The ArUcoDetectionNode class is a ROS2 node for detecting ArUco markers.

This class subscribes to image data, detects ArUco markers in the images, and publishes the pose of detected markers. It uses OpenCV and cv_bridge for image processing and marker detection.
