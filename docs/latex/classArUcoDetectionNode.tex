\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classArUcoDetectionNode}{Ar\+Uco\+Detection\+Node}} class is a ROS2 node for detecting Ar\+Uco markers. 

This class subscribes to image data, detects Ar\+Uco markers in the images, and publishes the pose of detected markers. It uses Open\+CV and cv\+\_\+bridge for image processing and marker detection.
